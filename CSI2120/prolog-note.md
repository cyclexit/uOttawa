## Function
* `length(?List, ?Int)`: get the list length.
* `member(?Elem, ?List)`: return true if *Elem* is a member of the *List*.

## Prolog Environment
### Load file
In Prolog environment, type `[file_name]` to load a file.
### Turn on/off trace.
* Turn on: `trace.`
* Turn off: `notrace, nodebug.`
